  • ML Model

    Crop Recommendation System

    Crop Recommendation System is a machine learning based web application that suggests the best crop to grow using soil nutrients and weather conditions. It analyzes inputs like nitrogen, phosphorus, potassium, pH, temperature, humidity, and rainfall to give accurate, data-driven recommendations. Built with Python and a Random Forest model, it delivers reliable predictions for different soil and climate types. The system is deployed as an easy-to-use web interface so farmers can get real-time crop suggestions.

  • AI & ML

    PDF Question Answering System

    PDF Question Answering System is an AI application that allows users to ask questions from uploaded PDF documents and receive context-based answers. It converts PDF text into embeddings, stores them in a vector database, and uses LangChain with an Ollama LLM to retrieve relevant content and generate accurate responses through semantic search.
